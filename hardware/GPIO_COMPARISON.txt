GPIO PIN ASSIGNMENT COMPARISON - DRV8833 Motor Drivers
=============================================================================

BEFORE (Old Configuration):
┌────────────────────────────────────────────────────────────────────────┐
│ LEFT DRV8833                    │ RIGHT DRV8833                        │
├────────────────────────────────────────────────────────────────────────┤
│ GP8  → AIN1 (Motor L1 Dir A)    │ GP4  → AIN1 (Motor R1 Dir A)        │
│ GP9  → AIN2 (Motor L1 Dir B)    │ GP5  → AIN2 (Motor R1 Dir B)        │
│ GP2  → BIN1 (Motor L2 Dir A)    │ GP6  → BIN1 (Motor R2 Dir A)        │
│ GP3  → BIN2 (Motor L2 Dir B)    │ GP7  → BIN2 (Motor R2 Dir B)        │
└────────────────────────────────────────────────────────────────────────┘

Issues: Non-sequential pins, less intuitive wiring


AFTER (New Configuration) - ✅ CURRENT:
┌────────────────────────────────────────────────────────────────────────┐
│ LEFT DRV8833                    │ RIGHT DRV8833                        │
├────────────────────────────────────────────────────────────────────────┤
│ GP2  → AIN1 (Motor A Dir/PWM)   │ GP6  → AIN1 (Motor C Dir/PWM)       │
│ GP3  → AIN2 (Motor A Dir/PWM)   │ GP7  → AIN2 (Motor C Dir/PWM)       │
│ GP4  → BIN1 (Motor B Dir/PWM)   │ GP8  → BIN1 (Motor D Dir/PWM)       │
│ GP5  → BIN2 (Motor B Dir/PWM)   │ GP9  → BIN2 (Motor D Dir/PWM)       │
│                                 │                                      │
│ GP15 → nSLEEP (Enable)          │ GP17 → nSLEEP (Enable)              │
└────────────────────────────────────────────────────────────────────────┘

Benefits:
✅ Sequential GPIO pins (2-5, 6-9)
✅ Easier wiring and troubleshooting
✅ Matches problem statement specification
✅ Frees GP0/GP1 for GPS UART
